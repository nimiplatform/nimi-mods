# Pipeline Contract

> Owner Domain: `RL-PIPE-*`

## RL-PIPE-001 Canonical Simulation Chain

The required simulation chain is:

1. `generate-fact-graph`
2. `generate-decision-tree`
3. `render-screenplay`

## RL-PIPE-002 Information Sealing

Default mode is sealed information; omniscient mode is explicit opt-in.

## RL-PIPE-003 Sharing Pipeline

Share flow must include anonymization, preview confirmation, and signed publish.

## RL-PIPE-004 Event Traceability

Scenario/model/prompt versions must remain traceable in persisted artifacts.
