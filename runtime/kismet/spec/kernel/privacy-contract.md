# Privacy Contract

> Owner Domain: `KIS-PRI-*`

## KIS-PRI-001 Raw Birth Input Locality

Raw birth date, time, and place remain local to the mod by default.

## KIS-PRI-002 Local Consent Policy

Consent rules are authoritative in `tables/privacy-consent-policies.yaml`.
Local derived-profile persistence and local matching use MUST be explicit toggles.

## KIS-PRI-003 Derived Profile Scope

Reusable match profiles may only contain derived compatibility fields.

## KIS-PRI-004 Revocation

When local persistence consent is revoked, the mod MUST allow removal of saved local profiles from match selection.
