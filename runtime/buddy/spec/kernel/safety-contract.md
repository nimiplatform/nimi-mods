# 安全合约

> Owner Domain: `BD-SAFE-*`

## BD-SAFE-001 System Prompt 儿童安全指令

System prompt 必须包含以下强制指令：

1. 角色使用简单、温暖、积极的语言
2. 禁止讨论暴力、恐怖、色情或任何不适合儿童的内容
3. 禁止提供医疗、法律或财务建议
4. 遇到敏感话题时引导转向积极方向
5. 不模拟或鼓励危险行为

System prompt 的安全指令部分不可被用户自定义内容覆盖。

## BD-SAFE-002 回复长度限制

单次助手回复不超过 150 个字符（中文）或 80 个词（英文），适配儿童注意力跨度。

超长回复由 prompt 指令约束，不在 mod 层做硬截断。

## BD-SAFE-003 连续使用提醒

Buddy 追踪本次会话的连续对话时长：

1. 每 30 分钟显示一次休息提醒（非阻断性提示）
2. 提醒文案由角色以对话方式表达（如："我们休息一下眼睛吧！"）
3. 用户可以忽略提醒继续对话
4. 计时器在用户 5 分钟无操作后自动重置

## BD-SAFE-004 模型视觉安全

Live2D 模型视觉内容必须适合儿童：

1. 不包含暴露或暗示性内容
2. 不包含恐怖或惊悚元素
3. 模型切换时 UI 预览需展示完整角色外观
4. 自制模型必须通过视觉安全审查后才能上架

## BD-SAFE-005 数据最小化

Buddy 不收集或存储可识别儿童个人信息：

1. 对话历史仅存储在本地宿主持久化（Buddy 专属 sqlite），不上传
2. 语音输入仅用于实时转写，不持久化音频文件
3. 不记录用户年龄、姓名、位置等个人信息
4. 卸载 mod package 默认不删除数据；用户显式执行数据清理后才删除本地持久化内容
