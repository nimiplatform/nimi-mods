# Live2D Cubism Core

`live2dcubismcore.min.js` is not committed to this repository.

Download it yourself from the official Live2D Cubism SDK for Web page after reviewing the license terms:

- https://www.live2d.com/sdk/download/web/

Then place the file at:

- `nimi-mods/runtime/buddy/assets/live2d-core/live2dcubismcore.min.js`

Buddy loads this file locally at runtime and does not fall back to a CDN.
