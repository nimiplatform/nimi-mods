# Daily Fortune Contract

> Owner Domain: `KIS-DAY-*`

## KIS-DAY-001 Day Context Source

Date, timezone, and today GanZhi MUST be derived deterministically from local day context.

## KIS-DAY-002 Narrative Scope

Daily fortune prompts may explain the current day context but MUST NOT recompute pillars or five-element ratios.

## KIS-DAY-003 Availability Gate

Daily fortune generation MUST be blocked until a canonical profile exists.
