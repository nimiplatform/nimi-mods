# Acceptance Contract

> Owner Domain: `N-ACC-*`

## N-ACC-001 Table-Driven Cases

Acceptance cases are authoritative in `tables/acceptance-cases.yaml`.

## N-ACC-002 Minimum Coverage

Coverage must include approved path, adjusted path, reject path, context-missing path, and initiative cooldown no-op path.

## N-ACC-003 Verification Commands

All checks must pass:

1. kernel docs generation
2. docs drift check
3. consistency check
